# DirectAim
DirectAim by Aaron Merritt: A high-precision, absolute-positioning spatial gun. Using a Teensy 4.0 &amp; BNO086 IMU, it maps aim 1:1 to any screen without IR bars or cameras. Display-agnostic (laptop to projector) &amp; driverless. Housed in a custom Halo Pistol shell with a tactile limit-switch trigger. Real arcade feel, zero setup hassle.
